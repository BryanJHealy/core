Net Files - .net.xml

This file contains the definition of the intersection itself. This includes node and edge files. Net files should not be manually editted. They should be generated by either calling node and edge files or using NETCONVERT on an .osm file(from OpenStreetMaps).

The XML Schema to call node and edge files is as follows:
netconvert --node-files=MyNodes.nod.xml --edge-files=MyEdges.edg.xml \
  --output-file=MySUMONet.net.xml

If you also include connections and types the call is:
netconvert --node-files=MyNodes.nod.xml --edge-files=MyEdges.edg.xml \
  --connection-files=MyConnections.con.xml --type-files=MyTypes.typ.xml \
  --output-file=MySUMONet.net.xml
